Welkom. Platform-admin draagt de complete operationele keten: cluster,
identity, storage, security, observability. Deze module brengt alleen het
kader op orde — de Practitioner- en Expert-modules gaan dieper.

## Twee omgevingen, één codebase

| Omgeving | Hostnaam | Hoe |
|---|---|---|
| **Lokaal (k3d)** | `*.uwv-platform.local` | `make cluster && make bootstrap && make deploy-platform` |
| **Cloud (AKS)** | `*.uwv-platform.cloud` | dezelfde manifests, ander ingress + oauth2-proxy |

De portal-applicatie is een statische build die op beide ingressen draait —
client-side wordt `.local` herschreven naar `.cloud` waar nodig. Zie
[Layout.astro](/portal/src/layouts/Layout.astro) voor de rewrite-script.

## Componenten die alleen jij beheert

| Component | Wat doe jij hier? |
|---|---|
| **Keycloak** | Realm `uwv`: users, groups, federatie, MFA |
| **oauth2-proxy** | SSO-laag voor alle ingressen |
| **cert-manager** | TLS-certificaten (Let's Encrypt op AKS, mkcert lokaal) |
| **Stackable operators** | NiFi/Kafka/Spark/Hive/Trino/Airflow/Superset/OPA/ZK |
| **MinIO root** | bucket-creation; quota; versioning |
| **OPA bundle** | Reload na role-mapping-PR-merge |

## Observability — drie niveaus

```
1. Prometheus job=*    → "draait het?"
2. OpenSearch logs     → "wat zegt de pod?"
3. Stackable CR status → "wat denkt de operator?"
```

Een incident is pas geanalyseerd als alle drie consistent zijn. Een
Prometheus-up=1 met een failing reconcile in Stackable is geen
"draaiend" cluster.

## Break-glass

Voor noodsituaties (uitval, beveiligingsincident) is er een tijdelijke
break-glass-rol die meer toegang krijgt dan een normale platform-admin —
inclusief sensitive.* en audit-bypass voor herstel-acties. Voorwaarden:

- Activatie via een tweede admin (4-eyes);
- Maximaal 1 uur, daarna automatisch deactivatie;
- Alle handelingen worden hard gelogd en *binnen 24 uur* gereviewed door
  de data-steward + security-officer.

Procedure staat in [docs/runbook.md](/docs/runbook.md). Lees het voordat
je het ooit nodig hebt — niet pas op het moment zelf.
