In deze module lever je één onderzoek aan dat een ander over een jaar nog
kan reproduceren. Dat klinkt simpel; het is de moeilijkste discipline op
het platform.

## Het pakket

Eén PR met:

```
research/
  uc09_uitkomst_per_groep_2025q3/
    README.md                  # rapport met conclusies + caveats
    notebook.ipynb             # met as_of_date hardcoded
    fingerprint.json           # dataset-hash ten tijde van publicatie
    schema.yaml                # welke gold/sandbox-tabellen + dbt-tag
```

## README.md (rapport)

Structuur:

```markdown
# UC-09: Re-integratie-uitkomst per groep, Q3 2025

## Onderzoeksvraag
Bestaat er een verschil in re-integratie-uitkomst tussen
{omschrijving} groepen, gecorrigeerd voor leeftijd en uitkeringsoort?

## Bron
- Tabellen: sandbox.research.cohort_2025
- dbt-tag: research_2025_q3
- as_of_date: 2025-09-01
- Pseudonimisering: pseudo-v3

## Methode
[korte beschrijving — chi-kwadraat, regressie, etc.]

## Resultaten
[tabellen + chart-references]

## Conclusies
1. ...
2. ...

## Caveats
- k-anonimiteit: groep B&lt;10 samengevoegd met B'.
- Klein cohort, herhalen aanbevolen op grotere dataset.
- Glossary: {term} gebruikt zoals gedefinieerd in OpenMetadata, link.

## Reproduceren
$ jupyter notebook notebook.ipynb
$ python -c "from check import verify_fingerprint; verify_fingerprint()"
```

## fingerprint.json

```json
{
  "tables": ["sandbox.research.cohort_2025"],
  "as_of_date": "2025-09-01",
  "dbt_tag": "research_2025_q3",
  "row_count": 5740,
  "column_hashes": {
    "uitkeringsoort": "sha256:8a7d...",
    "leeftijdsklasse": "sha256:1f3c...",
    "re_integratie_uitkomst": "sha256:9e2a..."
  }
}
```

Een herhaalde run van het notebook met dezelfde `as_of_date` moet
dezelfde `column_hashes` opleveren. Verschil = data is gewijzigd, en dat
moet je weten.

## Verificatie

De auto-check valideert dat de PR de bovenstaande structuur heeft, dat
het notebook een vast `as_of_date` heeft en dat een
fingerprint-verificatie groen is.
